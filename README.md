# STM32 EV Telemetry System 

A miniature electric vehicle telemetry system inspired by embedded systems used in current cars. 

## Project Overview

This project will simulate a vehicle telemetry system by collecting sensor data, transmitting the date and displaying it through a Python dashboard. 

## Technologies Used

### Software
- Python
- VS Code
- Git/GitHub

### Hardware
- STM32 microcontroller
- Sensors
- Breadboard prototyping

## System Architecture 

Sensors -> STM32 -> UART -> Python Dashboard -> Data Visualization(CSV File)
